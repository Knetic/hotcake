Along with hotloading single classes, Hotcake is prepared to hotload classese that use inheritance.

The below "pulsate" button and example element are both controlled by a "view" class, same as in the last example.
However, this time the view class is subclassed into a SlidePulsateView, inheriting PulsateView.
If we make changes to the SlidePulsateView, and then hotload those changes, they show up as expected.
