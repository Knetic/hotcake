This example finally takes advantage of what sets Hotcake apart from other hotloaders - its ability to hotload classes.

The below "pulsate" button and example element are both controlled by a "view" class.
This view class initially pulses the square rapidly green. But what if we want to change that behavior?
The second version of the class pulsates it slowly red.
Pressing the "hotload" button will swap the first version of the script for the second version.
